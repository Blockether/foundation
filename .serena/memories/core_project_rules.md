# Core Project Rules

## Absolutely NO Code of Conduct
- Never create CODE_OF_CONDUCT.md files
- We are grown adults who expect mutual respect
- No behavioral rules or guidelines documents needed
- Focus on technical solutions and direct communication

## Code Quality Standards
- Always remove unused imports, fixtures, and code
- Maintain type safety with full annotations
- Clean up after yourself - no technical debt
- Follow Python best practices with modern tooling

## Communication Style
- Direct, professional, technical focus
- Assume competence and good intentions
- No excessive politeness or corporate speak
- Get to the point and solve problems

## Development Philosophy
- Foundation library for Agno-powered AI agents
- Modular, type-safe, clean architecture
- Minimal dependencies, maximum functionality
- Build for other developers to build upon
